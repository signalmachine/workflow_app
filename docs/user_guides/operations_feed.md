# Operations Feed

Date: 2026-03-31
Status: Active
Purpose: explain how to use the operations feed as a one-way coordinator and system status surface.

## 1. Open the feed

Open `/app/operations-feed`.

Use the feed to review the current request, proposal, and approval truth in one place.

## 2. What to look for

The feed should help you confirm:

1. recent request activity
2. recent proposal activity
3. recent approval activity
4. direct links into the relevant detail pages

## 3. Verify continuity

When something looks important in the feed, follow the link into the underlying workflow page and confirm:

1. the request reference matches
2. the proposal or approval record matches
3. the browser feed and the detail page agree on the same facts

## 4. Troubleshooting

If the feed is empty:

1. confirm the org session is active
2. confirm there are recent workflow records in the database
3. refresh the page once

If the feed shows stale information:

1. open the underlying detail page
2. compare the current record state
3. refresh the feed after confirming the record chain
